# Architecture

- Mise en place d'un serveur python répondant à un CLI en ligne de commande, les deux étant connectés en http.
- Les sources sont localisés dans le dossier ./src
- Chaque module indépendant doit être placé dans un dossier ./src/<module>

## Module simphonia

- Ce module est le serveur principal
- spécifications techniques dans ./documents/simphonia.md

## Module simcli

- Ce module est le CLI en ligne de commande permettant de piloter le serveur
- spécifications techniques dans ./documents/simcli.md
